# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes telecom customer churn data and builds an interactive Power BI dashboard to identify churn patterns.

## Objective
To understand customer behavior and identify key factors influencing churn.

## Tools Used
- Power BI
- CSV Dataset

## Key Insights
- Month-to-month contracts have highest churn rate.
- Long-term contracts reduce churn.
- Higher monthly charges increase churn probability.

## Files Included
- cleaned_churn_data.csv

## Outcome
This dashboard helps businesses improve customer retention strategies.
